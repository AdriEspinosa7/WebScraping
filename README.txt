# Instrucciones para ejecutar el scraper de bolsa

¡Hola!

Este programa se ejecuta con Docker.

## En Linux:

1. Abre la carpeta donde descomprimiste el programa.
2. Haz doble clic en el icono "Ejecutar Scraper Bolsa".
3. Se abrirá una terminal y empezará a ejecutarse el scraper.
4. La primera vez puede tardar un poco mientras descarga todo.
5. El scraper se ejecutará automáticamente todos los días a las 20:00 horas.

## En Windows:

1. Abre la carpeta donde descomprimiste el programa.
2. Haz doble clic en el archivo "run.bat".
3. Se abrirá una ventana de comandos y empezará a ejecutarse el scraper.
4. La primera vez puede tardar un poco mientras descarga todo.
5. El scraper se ejecutará automáticamente todos los días a las 20:00 horas.
